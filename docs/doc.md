## Tasks
Our framework supports three kinds of tasks, which can be specified by `--task option` with options below.

| Options   | Task                   |
|-----------|------------------------|
| attribute | Attribute modification |
| head_pose | Head pose modification |
| landmark  | Landmark modification  |
